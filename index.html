PC

  1. 1280 x 1024 해상도를 기준으로 한다.
  2. 메인 페이지의 경우 가로 1100px 이내, 세로 가변.
  3. 화면정렬 : 중앙정렬을 기본으로 한다.

TABLET

  1. 1024 x 768 해상도를 기준으로 한다.
  2. 메인 페이지의 경우 가로 825px 이내, 세로 가변.
  3. 화면정렬 : 중앙정렬을 기본으로 한다.

MOBILE

  1. 640 x 480 해상도를 기준으로 한다.
  2. 메인 페이지의 경우 가로 600px 이내.
  3. 화면정렬 : 중앙정렬을 기본으로 한다.
